# 快速开始 - 喝水提醒应用

## 最简单的方式：使用Android Studio

### 步骤1：安装Android Studio
1. 下载：https://developer.android.com/studio
2. 安装时选择"Standard"（推荐）
3. 安装完成后启动Android Studio

### 步骤2：打开项目
1. 点击 "Open an Existing Project"
2. 选择 `c:\drink_water` 文件夹
3. 等待Gradle同步完成（首次需要3-5分钟）

### 步骤3：运行应用
**使用模拟器（不需要手机）：**
1. 点击右上角的 "Device Manager"
2. 点击 "Create Device" → 创建一个新设备
3. 选择手机型号（如 Pixel 6）
4. 下载系统镜像（点击Download）
5. 点击绿色的 ▶️ Run按钮

**使用真实手机：**
1. 在手机上开启"开发者选项"
2. 开启"USB调试"
3. 用USB线连接手机
4. 点击绿色的 ▶️ Run按钮

## 生成APK安装包（供手机安装）

### 准备工作
1. 确保已安装Android Studio（至少需要Android SDK）
2. 确保已安装JDK 17或更高版本

### 构建APK
1. 双击运行 `build.bat`
2. 等待构建完成（首次构建需要5-10分钟）
3. 构建成功后，APK位于：
   `app\build\outputs\apk\debug\app-debug.apk`

### 安装到手机
1. 将 `app-debug.apk` 发送到手机（微信、QQ、数据线都可以）
2. 在手机上点击APK文件
3. 允许安装未知来源的应用
4. 等待安装完成

## 应用使用

### 首次使用
1. 打开应用，允许通知权限
2. 点击"我喝了一杯水"记录喝水
3. 点击右上角 ⚙️ 设置提醒

### 提醒设置
- 默认：每小时提醒一次，9:00-22:00
- 可以自定义提醒间隔（15-180分钟）
- 可以设置提醒时间段
- 关闭提醒则不会收到通知

## 常见问题

**Q: Gradle同步失败？**
A: 检查网络连接，或删除项目中的`.gradle`文件夹后重新打开

**Q: 找不到Android SDK？**
A: 打开 `local.properties`，将 `YourUsername` 改成你的Windows用户名

**Q: 构建失败提示JAVA_HOME错误？**
A: 安装Android Studio会自动配置，或者手动安装JDK 17并设置JAVA_HOME环境变量

**Q: 模拟器很慢？**
A: 在Android Studio的SDK Manager中安装Intel HAXM（Intel CPU）或使用手机

**Q: 手机上收不到通知？**
A: 检查应用的通知权限，确保没有在电池优化中禁用
