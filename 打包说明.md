# 心水 App 安卓打包说明

## 📱 响应式适配

已完成以下响应式设计：
- 自动适配不同屏幕尺寸（320px - 428px 宽度）
- 在手机上全屏显示
- 移除桌面端固定框架
- 统计卡片在小屏幕上自动调整布局

## 🎨 App 图标生成

### 方法一：使用在线转换（推荐）

1. 打开 `icon-generator.html` 文件
2. 点击"下载全部"按钮
3. 将下载的图标文件（icon-48.png 到 icon-512.png）保存到项目根目录

### 方法二：手动生成

访问在线图标生成器：
- https://realfavicongenerator.net/
- 上传 `icon-192.svg` 文件
- 下载生成的图标包，将 PNG 文件复制到项目根目录

## 📦 打包为安卓应用

### 方法一：使用 Capacitor（推荐）

1. 安装 Node.js（如果未安装）
2. 安装 Capacitor CLI：
   ```bash
   npm install -g @capacitor/cli
   ```

3. 初始化 Capacitor 项目：
   ```bash
   cd c:\drink_water
   npx cap init "心水" com.drinkwater.app
   ```

4. 安装安卓平台：
   ```bash
   npm install @capacitor/android
   npx cap add android
   ```

5. 同步文件：
   ```bash
   npx cap sync
   ```

6. 打开安卓项目：
   ```bash
   npx cap open android
   ```

7. 在 Android Studio 中构建 APK：
   - Build > Build Bundle(s) / APK(s) > Build APK(s)
   - 生成的 APK 位于 `app/build/outputs/apk/` 目录

### 方法二：使用 PWA Builder

1. 访问 https://www.pwabuilder.com/
2. 上传 `web-preview.html` 文件
3. 选择 "Android"
4. 下载生成的 APK

1. 在手机上启用"未知来源应用"安装
2. 下载 APK 文件到手机
3. 点击 APK 文件进行安装
4. 打开"心水"应用开始使用！
